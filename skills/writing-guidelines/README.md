# Writing Guidelines Skill

This directory captures editorial guidance and review criteria for repository content.

Usage
- Follow these guidelines when authoring new skills, knowledge articles, patterns, recipes, playbooks, examples, references, rules, checklists, and prompts.
- Use `content-template` templates as the structural foundation.
- Use this skill to review documents for clarity, consistency, and adherence to repository conventions.

Core rules
- Use active voice and professional tone.
- Prefer concise, direct sentences.
- Use headings to break large topics into manageable sections.
- Define acronyms and technical terms on first use.
- Keep examples focused and contextualized.
- Document assumptions, inputs, and expected outputs clearly.

Review guidance
- Confirm the file uses the correct template type.
- Ensure purpose, prerequisites, and scope are clearly defined.
- Validate that claims are supported by references or evidence.
- Check for consistent formatting and section naming.
- Verify any procedural content includes explicit validation steps.
