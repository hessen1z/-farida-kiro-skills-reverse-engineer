# Security Mitigations Example

## Scenario
Inspect a Windows binary to identify enabled exploit mitigations and runtime protections.

## Steps
1. Check the PE header for ASLR, DEP, and CFG support.
2. Verify process mitigation state in a debugger or system tool.
3. Document protections that may affect exploitation or analysis.

## Outcome
- Confirm which mitigations are active for the binary.
- Understand how the mitigations influence runtime behavior.
- Provide evidence for security and analysis planning.
