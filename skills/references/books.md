# Books

## Purpose
This page collects core books that are useful for understanding Windows internals, binary analysis, reverse engineering, and malware triage.

## Recommended References
- Windows Internals — a foundational reference for system architecture, kernel behavior, and debugging context.
- Practical Malware Analysis — a strong introduction to malware triage and analysis workflows.
- Reversing — a practical guide to disassembly, analysis methodology, and reverse engineering techniques.

## Practical Guidance
- Use these books as primary references when the repository needs stronger conceptual grounding.
- Prefer the books that align best with the current task rather than consuming everything at once.
- Cross-reference the relevant repository skills when a section or chapter maps directly to analysis workflows.
