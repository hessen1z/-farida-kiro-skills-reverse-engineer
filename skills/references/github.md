# GitHub Resources

## Purpose
This page tracks public GitHub projects and repositories that are valuable for tooling, experimentation, and reference implementations related to reverse engineering and analysis.

## Useful Categories
- Disassembly and debugging tools such as IDA, Ghidra, and related plugins.
- Instrumentation and hooking libraries such as MinHook and PolyHook.
- Example projects that demonstrate pattern scanning, PE parsing, or runtime instrumentation.

## Practical Guidance
- Review the repository’s purpose and maintenance status before relying on it as a primary reference.
- Prefer projects that include documentation, examples, or a clear API contract.
- Link back to the relevant repository skills or examples when a project maps clearly to a documented workflow.
