# Kernel Analysis Prompt

- Identify the Windows kernel subsystem or driver component under analysis.
- Map the relevant kernel objects, dispatch paths, and IRQL boundaries.
- Determine whether the observed behavior arises from scheduler, memory, or driver issues.
- Recommend the debugger commands and evidence needed to verify the diagnosis.
