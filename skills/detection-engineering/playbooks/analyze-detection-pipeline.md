# Analyze Detection Pipeline

## Purpose
Guide the review of an existing detection pipeline from telemetry collection through alerting and tuning.

## Workflow
1. Map the complete event path from collection to alerting.
2. Check the telemetry source, schema, and enrichment.
3. Review rule logic and correlation quality.
4. Validate signal quality and performance.
5. Recommend tuning and follow-up work.
