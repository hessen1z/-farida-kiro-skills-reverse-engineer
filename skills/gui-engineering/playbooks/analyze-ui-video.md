# Playbook: Analyze UI Video

## Purpose
Infer the structure and interaction model of a UI from a video recording.

## Steps
1. Observe the interface across key moments in the video.
2. Capture the main navigation, window transitions, and state changes.
3. Identify recurring components and interaction patterns.
4. Record any animated or dynamic elements that matter for the final implementation.
5. Translate the observed behavior into a reconstruction plan.

## Output
A behavior-oriented UI analysis describing layout, interaction flow, and implementation priorities.
