# Recipe: Clone Professional UI

## Goal
Recreate the feel and structure of a polished desktop application UI using Dear ImGui while keeping the implementation maintainable.

## Procedure
1. Study the reference interface and define the main visual hierarchy.
2. Identify which portions must be exact, which can be simplified, and which should be adapted for usability.
3. Implement a layout skeleton with reusable widgets.
4. Add visual polish with themed panels, cards, and DrawList effects.
5. Review the final result against the reference and quality checklist.

## Output
A convincing reproduction of a professional desktop UI that balances fidelity, maintainability, and implementation effort.
