# Architecture Patterns

## Overview
Architecture patterns describe proven structural approaches to designing large systems and software platforms.

## Patterns Covered
- layered architecture
- clean architecture
- event-driven architecture
- microkernel and service-oriented design

## Practical Usage
- choose the right pattern for the domain and quality goals.
- apply architectural rules consistently to avoid hidden coupling.
- document the intended pattern and service boundaries.
