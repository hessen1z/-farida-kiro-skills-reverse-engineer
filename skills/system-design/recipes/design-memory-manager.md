# Design Memory Manager Recipe

## Goal
Design a memory manager that provides controlled allocation, tracking, and deallocation for C++ systems.

## Steps
1. Define allocator interfaces for different allocation classes.
2. Implement pools or arenas for predictable allocation behavior.
3. Track allocation metadata for debugging and leak detection.
4. Provide safe deallocation and optional bounds checking.

## Focus Areas
- ownership and lifetime
- fragmentation and allocation efficiency
- debugging instrumentation
- alignment and platform support
